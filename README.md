# Табло соревнования

Локальное приложение для управления результатами и трансляции в формате 1920×1080. Интернет и Excel не требуются: все данные вводятся в админке и сохраняются в `data/scoreboard.json`.

## Запуск

1. Дважды щёлкните `start.bat`.
2. Откройте `http://localhost:3000/admin`.
3. При необходимости отредактируйте организации и участников.
4. Во вкладке «Выбор участников» заполните теорию, отметьте до трёх участников практики и до двух участников финала от каждой команды.
5. Во вкладке «Практика» заполните 13 критериев выбранных участников.
6. Во вкладке «Финал» отдельно заполните две ноги каждого выбранного финалиста.
7. Нажмите «Сохранить изменения» - эфирные экраны обновятся автоматически.

Эфирные страницы:

- `/theory` - теоретический этап;
- `/practice` - практический этап;
- `/final` - результаты финального этапа;
- `/admin` - организации, участники и результаты.

Перед каждым сохранением создаётся резервная копия JSON в `data/backups`.

Правила подсчёта:

- теория - весь состав команды, до 60 баллов на участника и до 300 на команду;
- практика - три выбранных участника, до 100 баллов на каждого и до 300 на команду; «Допы» входят в эти 100 баллов;
- финал - два выбранных участника, две ноги по 100 баллов у каждого, до 400 на команду;
- места в финале определяются только финальными баллами, без добавления теории и практики.

Старый файл данных можно привести к актуальному формату командой `npm.cmd run migrate`. Перед миграцией создаётся резервная копия.

## Разработка

Требуется Node.js 22 или новее.

```powershell
npm.cmd install
npm.cmd run dev
```

Vite работает на `http://localhost:5173`, API - на `http://localhost:3000`.

Проверка и production-сборка:

```powershell
npm.cmd run check
npm.cmd run build
```

Создание переносимого офлайн-комплекта:

```powershell
npm.cmd run release
```
